# Portfolio

Single-page portfolio built with React, Vite, and Tailwind CSS. Black background, white text, minimal layout with four sections: Hero, About, Projects, and Contact.

## Prerequisites

- Node.js 18+ and npm

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for production

```bash
npm run build
```

Output is in the `dist` folder. To preview the production build locally:

```bash
npm run preview
```

## Replace placeholders

- **Contact**: Update the email and LinkedIn URLs in `src/App.jsx` (Section 4) with your real address and profile.
- **Third project**: Replace "Project Title Placeholder" and the description in the third project card with your actual project.
- **Images**: Replace the gray placeholder divs in the project cards with `<img>` tags or your preferred image components when you have assets.
